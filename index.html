is this client good?

hopefully! you can make it better by sending feedback to the developer, but ideally this client should be simple enough to navigate and be packed with as many features as possible. for now, it's still in development so not yet on that one sadly.

how secure will my info be?

it depends on the server and socket URLs you type. the official maelink servers (which are default) hash your info, and this client uses the best and industry-standard security practices. it's all up to you and whether the servers hash your data.
